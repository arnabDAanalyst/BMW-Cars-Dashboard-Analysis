● Top Revenue-Generating Model

● Average Price by Fuel Type
● Average Price by Transmission Type

● Average Engine Size by Region

3. Sales Performance
● High vs. Low Sales Classification Count
● Average Sales Volume by Region
● Sales Volume by Fuel Type
● Sales Volume by Transmission Type

4. Vehicle Characteristics & Demand
● Engine Size Impact on Sales
(Compare average sales volume by Engine_Size_L range)

(Do lower mileage cars have higher sales volume?)
